How to get your Square credentials
  1. 1
    Create a Square Developer account

    Go to developer.squareup.com and sign in with your existing Square account (the same one you use for payments at the salon).

  2. 2
    Create an Application

    Click "+ New Application", give it a name like "Chaltu's Salon Booking", then open it. You'll find your Application ID on the Credentials tab — copy it into the field above.

  3. 3
    Get your Location ID

    In the same app, go to the Locations tab. Copy the Location ID for your salon location (starts with L) and paste it above.

  4. 4
    Add your Access Token to Railway

    The Access Token (also on the Credentials tab — never share this) must be added as a Railway environment variable named SQUARE_ACCESS_TOKEN. This is used server-side to actually charge the card. Do not paste it here.

  5. 5
    Test with Sandbox first

    Keep the environment set to Sandbox and use Square's test card 4111 1111 1111 1111 to make sure the deposit flow works end-to-end before switching to Production.

The Application ID and Location ID are safe to store — they identify your app but cannot be used to move money. Your Access Token is the sensitive secret; keep it only in Railway environment variables and never commit it to code.
